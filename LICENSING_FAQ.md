# Licensing FAQ

## General Questions

### What license is this node released under?
This node is licensed under the Business Source License 1.1 (BSL 1.1).

### Can I use this node for free?
Yes, for personal use, educational purposes, research, and internal business use that does not involve resale or commercialization.

### When do I need a commercial license?
A commercial license is required when using this node in any revenue-generating context, including SaaS platforms, managed services, or commercial products.

## Commercial Use

### Is internal SaaS infrastructure considered internal use?
No. If the infrastructure supports a revenue-generating service, it is considered commercial use.

### What if customers never see the node?
Visibility is irrelevant. Revenue enablement constitutes commercial use.

### What if we are a startup?
Startups are commercial entities and require a commercial license for production use.

### What if we only use one workflow?
Scope does not change license requirements. Any commercial use requires a license.

### Do proof-of-concepts require a license?
Internal, non-customer-facing POCs are permitted. Production SaaS deployments require a license.
